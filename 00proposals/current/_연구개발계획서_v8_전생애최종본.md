## 1. 연구개발의 필요성 (Why the project is necessary)

### 1-1. 전생애 인간 발달과 이해의 필요성

- **영유아·아동기**: 보건복지부(2024) 조사에 따르면 팬데믹 이후 영유아의 언어·사회성 발달지연 사례가 전년 대비 18.6% 증가하였다. 초기 실행기능 및 인지 발달은 인간의 이후 적응과 정서적 정체성 형성의 핵심이다.
- **청소년기**: 질병관리청 청소년건강행태조사(2023)는 우울 경험률이 26%, 자살사고율 13.5%로 나타났다. 이는 단순히 보건적 문제를 넘어, 자아 정체성 확립 과정에서의 심리적-신경생물학적 불균형을 인간 발달적 관점에서 깊이 이해해야 함을 시사한다.
- **청년∼중년기**: 고용노동부(2024) 직장인 정신건강 실태조사에서 20-39세 번아웃 지수는 62.3점(100점 만점)으로 2021년 대비 7.4점 상승했다. 중년층(40-59세)의 돌봄·경제 부담이 우울증 발병률을 1.8배 증가시킨다는 연구(서울연구원, 2024)도 보고되었다.
- **고령기**: 보건복지부 국민정신건강실태조사(2024)는 65세 이상 고독감 비율 34.1%, 주요 우울장애 유병률 6.3%를 기록했다. 치매 발병 전 단계에서 언어·시간지각 변화가 선행 지표가 될 수 있음에도 이에 대한 교육·개입은 부족하다.

### 1-2. 통합적 인간 이해 플랫폼의 부재

- 서울대 학부 교양과목 다수가 특정 생애 단계에 집중해 있으며, 전생애 관점을 기반으로 뇌-심리-언어 데이터를 통합적으로 다루며 '인간이란 무엇인가'를 탐구하는 과정이 없다.
- WHO(2023)는 생애주기별 맞춤 전략과 가족·지역사회의 협력, AI 및 디지털 기술의 책임 있는 활용을 강조하였다.
- Lancet Commission on Youth Mental Health(2024)와 Nature Cities(2024)는 인간의 적응과 웰빙을 **보호자·도시 시스템·지역사회**를 품은 다층적 이해의 산물로 정의한다. 이는 학부 수준에서 전생애적 인간 이해와 데이터 기반 문제 해결 역량을 길러야 함을 시사한다.

### 1-3. 베리타스 강좌 2 취지와의 정합성

- 베리타스 강좌 2는 실존적 삶의 주제를 중심으로 학제 간 탐구와 실천을 강조한다. 본 과목은 "삶과 죽음, 연결과 고립, 성공과 불안, 학습과 성장"이라는 실존 프레임을 영유아부터 노년까지 확장하여, 학생들이 **전생애에 걸친 인간 발달과 존재적 본질**을 통합적으로 이해하도록 설계했다.
- 서울대학교의 AI 인프라 확대와 연동해, AI-CoScientist 프레임워크를 수업 전 과정에 도입함으로써 인간의 과학적 탐구 한계를 확장한다.
- 이해연 교수의 Yale-NUS Lab Worksheet 모델과 차지욱 교수의 SwiFT(fMRI Transformer) 연구, 전현애 교수의 언어·시간지각 연구를 결합해, 이론-데이터-실천을 통합한 대표적 융합 교과목이 될 것이다.

## 2. 연구개발 목표 및 내용 (Aims of the project)

### 2-1. 교과목 목표

1. 전생애 관점에서 인간을 설명하는 **뇌-행동-의미(Triangular Integration)** 모델을 학습한다.
2. AI-CoScientist 생태계를 활용하여 실제 신경영상·언어·사회 통계 데이터를 분석하고 근거 기반 인간 이해 모델을 설계한다.
3. 학생 주도 프로젝트를 통해 **아동·청소년·성인·고령** 각 단계의 존재적/사회적 문제를 진단하고 해결 전략을 제안한다.

### 2-2. 교과목 주제와 주요 내용

- 주제: "전생애 인간 이해와 실존적 삶의 주제를 AI와 데이터로 탐구하기"
- 구성: 영유아 실행기능, 청소년 정체성·언어 서사, 성인 번아웃·디지털 표현형, 고령 고독·인지유지 등 생애주기 전반.
- 핵심 프레임워크: Wise Intervention(Walton & Crum, 2022), Design Thinking(Yeager et al., 2016), Lab Worksheet(이해연), AI-CoScientist(Lu et al., 2024).

### 2-4. 강의 계획 (Flipped Learning 기반 상세 설계)

#### 수업 구조: Flipped Learning + Team-Based Learning 통합

본 교과목은 **완전한 플립 러닝** 모델을 채택한다. 학생들은 수업 전 마이크로 영상(5분×2-3개)과 필독 자료를 학습하고, iRAT(개인 준비도 테스트)를 완료한 후 수업에 참여한다. 수업 시간(150분)은 전적으로 **팀 활동, 데이터 실습, 토론**에 할애된다.

**수업 시간 구조 (150분)**:
- [0-15분] tRAT(팀 준비도 테스트) + 오개념 교정 미니강의
- [15-75분] 팀 Application 활동 (4S 원칙: Same problem, Significant, Specific choice, Simultaneous reporting)
- [75-90분] 휴식 + 비공식 교류
- [90-135분] 데이터 실습 / 프로젝트 작업 (Jupyter, Python, AI 도구)
- [135-150분] 성찰 저널 작성 + 다음 주 안내

#### 차지욱 교수 5강의 통합 서사: "예측하는 뇌, 데이터로 이해하는 인간"

차지욱 교수가 담당하는 5개 강의(1, 5, 8, 12, 13주차)는 **"예측(Prediction)"**이라는 통합 프레임워크로 연결된다. 이 서사 구조를 통해 학생들은 개별 강의를 고립된 주제가 아닌 하나의 coherent한 여정으로 경험한다.

```
┌─────────────────────────────────────────────────────────────────────────┐
│  1주: AI가 예측한다 → 뇌도 예측 기계 (Predictive Machine)               │
│       ↓                                                                 │
│  5주: 뇌가 행동을 예측한다 (SwiFT: Brain → Behavior Prediction)         │
│       ↓                                                                 │
│  8주: 행동이 정신건강을 예측한다 (Digital Phenotyping)                   │
│       ↓                                                                 │
│  12주: 뇌가 감정을 예측한다 (Predictive Coding & Constructed Emotion)   │
│       ↓                                                                 │
│  13주: 유전자가 뇌 예측 회로를 형성한다 (PGS → Brain → Behavior)        │
└─────────────────────────────────────────────────────────────────────────┘
```

**핵심 메시지**: 인간의 뇌는 끊임없이 예측하는 기계이며, AI는 이 예측 과정을 모방하고 증폭한다. 유전자는 예측 회로의 기초를 형성하고, 환경은 이 회로를 조율한다. 디지털 데이터는 이 예측 과정의 흔적을 담고 있다.

### 2-5. 학생 활동과 과제 (Flipped Learning 기반)

#### Pre-class 활동 (매주 수업 전)
- **마이크로 영상 시청**: 5분 × 2-3개, 총 10-15분 (영상 내 삽입 퀴즈 포함)
- **필독 자료**: 논문 Abstract/Methods 또는 보고서 요약 (10-15페이지 이내)
- **iRAT 완료**: 10문항 객관식, 수업 24시간 전 마감, 80% 이상 정답 시 보너스 점수

#### In-class 활동 (매주 150분)
- **tRAT (팀 준비도 테스트)**: 동일 10문항을 팀 토론 후 합의하여 제출, 즉각 피드백
- **팀 Application 활동**: 4S 원칙 적용, 실제 데이터 기반 문제 해결, 동시 발표
- **데이터 실습**: Jupyter Notebook 기반 분석, AI 도구 활용, 코드 작성 및 해석
- **성찰 저널**: "오늘 발견한 것 / 아직 모르는 것 / 다음 단계" 3문장 작성

#### 주요 과제 및 평가
- **Lab Worksheet (5회)**: 프로젝트 정의→데이터 준비→분석→개입 설계→확산 계획까지 단계별 점검
- **데이터 분석 과제 (4회)**: fMRI(SwiFT), 언어(LLM), 감정 구성 회로(vmPFC-공포 일반화), 유전-뇌-행동 경로(PGS) 데이터 세트 분석, 재현 가능한 Jupyter Notebook 제출
- **중간/최종 발표**: 중간(Week10) 팀별 개입 설계 발표, 최종(Week15) 전생애 정책 제안과 APA 논문 발표
- **동료 평가 (3회)**: Week 7, 10, 15에 팀원 기여도 평가, 성적에 10% 반영
- **정성 활동**: 지역 파트너 인터뷰, VR 공감 체험, 시니어 서사 인터뷰, 포커스 그룹

#### 평가 체계

| 구분 | 비중 | 세부 내용 |
|------|------|-----------|
| iRAT (개인 준비도) | 15% | 매주 사전 학습 확인, 자동 채점 |
| tRAT + 팀 활동 | 25% | 팀 준비도 테스트 + 팀 Application 활동 점수 |
