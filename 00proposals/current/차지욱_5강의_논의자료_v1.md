## 2. 5강의 통합 서사: "예측하는 뇌, 데이터로 이해하는 인간"

5개 강의는 **"예측(Prediction)"**이라는 프레임워크로 연결됩니다.

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   1주: AI가 예측한다                                                    │
│         → 뇌도 예측 기계다 (Predictive Machine)                         │
│                          ↓                                              │
│   5주: 뇌가 행동을 예측한다                                             │
│         → SwiFT: fMRI → Behavior Prediction                            │
│                          ↓                                              │
│   8주: 행동이 정신건강을 예측한다                                       │
│         → Digital Phenotyping: Behavior → Mental Health                │
│                          ↓                                              │
│  12주: 뇌가 감정을 예측한다                                             │
│         → Predictive Coding & Constructed Emotion                      │
│                          ↓                                              │
│  13주: 유전자가 뇌 예측 회로를 형성한다                                 │
│         → PGS → Brain → Behavior                                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**핵심 메시지**:
- 인간의 뇌는 끊임없이 예측하는 기계
- AI는 이 예측 과정을 모방하고 증폭
- 유전자는 예측 회로의 기초, 환경은 이 회로를 조율
- 디지털 데이터는 예측 과정의 흔적
